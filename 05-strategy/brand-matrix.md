# 品牌矩阵

## 1. 四种品牌定位

| 品牌类型 | 定位 | 价格区间 | 目标用户 | 溢价来源 |
|---------|------|---------|---------|---------|
| 高端品牌 | 品质溢价 | 均价×2以上 | 高净值用户 | 品质/设计/服务 |
| 大众品牌 | 主流市场 | 市场均价 | 大众消费者 | 性价比/知名度 |
| 性价比品牌 | 低价竞争 | 均价×0.7以下 | 价格敏感用户 | 供应链效率 |
| 白牌 | 无品牌溢价 | 成本定价 | 渠道商/企业客户 | 生产成本 |

## 2. 品牌矩阵推导逻辑

```
市场分析结论：品类规模 X 亿，集中度 CR3=45%
    ↓
自身能力评估：供应链优势 / 研发能力 / 资金储备 / 团队经验
    ↓
竞品格局判断：哪块是空白，哪块是红海
    ↓
品牌矩阵确定：单品牌打全场 / 多品牌分层覆盖
```

## 3. 单品牌 vs 多品牌

| 策略 | 适用条件 | 优势 | 劣势 |
|------|---------|------|------|
| 单品牌 | 资源有限 / 品牌力强 | 聚焦资源，品牌统一 | 难以覆盖多价格带 |
| 多品牌 | 资金充足 / 市场分层明显 | 各价格带精准卡位 | 管理成本高，品牌稀释风险 |

## 4. 战略矩阵制定检查清单

在确定矩阵之前，必须有数据支撑：

- [ ] 市场大盘规模 / 增长率 / 季节性数据
- [ ] Top 10 竞品价格带分布图
- [ ] 各价格带销量集中度分析
- [ ] 自身供应链成本结构清晰
- [ ] 自身资金储备能支撑的目标规模
- [ ] 团队运营能力评估（能做多少 SKU/店铺）
