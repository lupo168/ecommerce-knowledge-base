# 05 战略矩阵

> **前置条件**：必须先完成商业分析（市场大盘 + 竞品三维拆解），才能制定战略矩阵。拍脑袋定战略容易踩空。

## 三层矩阵

```
品牌矩阵  ←  品牌定位：高端/大众/性价比/白牌
    ↑
店铺矩阵  ←  渠道布局：旗舰/专卖/折扣/分销
    ↑
产品矩阵  ←  产品角色：引流/利润/形象/防御
```

从下往上推导：产品矩阵支撑店铺矩阵，店铺矩阵支撑品牌矩阵。

## 推导逻辑

```
市场分析结论（规模/增长/集中度）
        ↓
自身能力评估（供应链/研发/资金/团队）
        ↓
竞品格局判断（空白/红海/机会点）
        ↓
品牌→店铺→产品 矩阵确定
```
