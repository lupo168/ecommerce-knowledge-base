# 店铺矩阵

## 1. 常见店铺类型

| 店铺类型 | 定位 | 价格带 | 适用场景 |
|---------|------|-------|---------|
| 旗舰店 | 品牌形象 + 全品类覆盖 | 全价格带 | 品牌官方渠道 |
| 专卖店 | 专业品类 + 中高端 | 中高价格带 | 垂直品类深耕 |
| 折扣店 | 促销出货 + 库存清仓 | 低于日常价 | 临期/尾货处理 |
| 分销店 | 渠道下沉 + 价格覆盖 | 低价格带 | 多店铺铺货 |

## 2. 店铺矩阵推导逻辑

```
竞品分析结论
    ↓
品牌定位选择（高端/大众/性价比/白牌）
    ↓
渠道策略选择（直销/分销/混合）
    ↓
店铺矩阵确定
```

## 3. 自营矩阵 vs 分销矩阵

| 维度 | 自营为主 | 分销为主 |
|------|---------|---------|
| 适用阶段 | 品牌初期/爬坡期 | 品牌成熟/扩张期 |
| 利润率 | 高（无中间环节） | 低（让利给分销商） |
| 管控力 | 强（统一运营） | 弱（价格/服务参差） |
| 覆盖速度 | 慢（自建渠道） | 快（借用现成网络） |
| 风险 | 低（自主可控） | 高（串货/乱价风险） |
