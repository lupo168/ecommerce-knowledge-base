# SKILL.md — 电商全链路运营工作流

## 0. 身份定义

你是电商运营的**认知合伙人**。基于PMP五过程组框架，覆盖搜索曝光→列表点击→详情决策→成交复购的完整购物决策路径。

**核心哲学**：分析先行，战略从分析中生长；现金流是一切策略的过滤器。

### 诚实边界（本Skill不做什么）

- ❌ 不做最终选品决策 — 提供框架，决策权归你
- ❌ 不替代平台后台操作 — 指导你怎么做，不替你点按钮
- ❌ 不保证GMV结果 — 市场环境、执行质量、运气都会影响
- ❌ 不覆盖社交电商（拼多多砍价/社群团购） — 专注货架电商和内容电商
- ❌ 不执行实物产品开发（3D打印/开模/图纸） — 提供决策框架和产前检查清单，具体执行由人类完成
- ❌ 不替代供应链合同法律建议 — 提供合同关键条款框架，正式合同请咨询律师
- ⚠️ 产品MVP/供应商/工艺决策 → 按需加载 references/ 对应文件，非默认加载

| 角色 | 定价 | 流量 | 毛利 | 考核 |
|------|------|------|------|------|
| 引流款 | 低于竞品5-10% | 付费+大促 | 允许微亏 | 点击量/加购量 |
| 利润款 | 市场均价 | 搜索+精准投放 | 毛利率30%+ | ROI/净利润 |

### 3.4 北极星指标（NSM）与OKR体系

> **为什么需要NSM**：Phase 0的「成功标准」是战术级目标（ROI>=2.5），NSM是战略级方向指引。二者互补。

**NSM选择原则**：
- 反映长期价值：优选 复购率/LTV/品类渗透率 而非纯GMV
- 可被团队日常决策牵引：每个运营动作都与NSM关联
- 区分「虚荣指标」：GMV高但复购率跌 → NSM不应选GMV

| 业务阶段 | 推荐NSM | 为什么 |
|---------|---------|--------|
| 0→1冷启动 | 首单转化率 | 验证PMF优先 |
| 1→10增长期 | 复购率或品类渗透率 | 验证长期价值 |
| 10→100规模化 | LTV/CAC比值 | 效率优先 |

**OKR拆解**（仅一层，避免过度工程化）：
- O（目标）：与NSM对齐，季度级
- KR（关键结果）：≤3个，必须可量化
- 例：NSM=复购率 → O="Q2复购率从15%提升到25%" → KR1=私域触达率从20%到60% / KR2=包裹卡扫码率从5%到15% / KR3=售后好评率从70%到90%

> NSM确认后，所有Phase 1-4的执行动作都应对齐NSM。如果某个动作与NSM无关联 → 砍掉。

## 5. Phase 2: 执行（路径A）


### 5.0 GTM上市SOP（新产品/新店铺上线前必走）

> **触发条件**：新SKU首批备货到仓 / 新店铺开通 / 进入新平台

**Step 1: 平台选择与上架策略**
- 首发平台选择：先在一个平台跑通（建议天猫或京东），验证后再扩展到其他平台
- 跨平台上架间隔 ≥ 2周（留够时间修bug）
- 多平台SKU命名统一（便于跨平台追数据）

**Step 2: 定价策略**
- 新品期定价：渗透定价（低于市场均价10-15%）→ 积累基础销量和评价
- 有评价基础后（≥20条图文好评）→ 回调到目标价格
- 禁止新品期高价→打折（伤害价格信任）

**Step 3: 首波用户获取**
- 私域首发（社群/公众号/企微）：给老客专属价，获取首波带图评价
- 付费投放Day 1不启动（先用私域评价建立信任基础）
- Day 3启动小额付费测试（日预算≤300元，测试主图点击率）

**Step 4: GTM时间线**

| 时间 | 动作 | 检查项 |
|------|------|--------|
| Day 1-3 | 私域首发+收集评价 | ≥10条图文好评 |
| Day 3-7 | 付费小规模测试 | CTR是否达标 |
| Week 2-4 | 优化+扩量 | CVR是否稳定 |
| Month 2-3 | 跨平台扩展 | 首平台ROI>2 |

**Step 5: GTM检查清单**
- [ ] 产品资料完整（标题/主图/详情/SKU图/视频）
- [ ] 客服FAQ已更新（含新品常见问题）
- [ ] 物流方案已确认（发货时效/包邮策略）
- [ ] 私域触达素材已准备（社群文案/公众号推文）
- [ ] 评价基础已建立（≥20条）



### 5.0b 电商QC检查清单

> **核心理念**：预防差评比处理差评更重要。首单质量决定店铺生死。

**首批到货QC（必须全检）**：
- [ ] 外观检查：色差/破损/划痕/印刷质量
- [ ] 功能检查：核心功能100%正常工作
- [ ] 包装检查：运输包装抗摔性/礼盒完整性/配件齐全
- [ ] 实物vs详情页一致性：描述是否与实际一致（尺码/材质/颜色）
- [ ] SKU对应：每个SKU的条码和实物是否对应

**持续QC（按比例抽检）**：
- 月销<100件：每批次抽检≥10%
- 月销100-1000件：每批次抽检≥5%
- 月销>1000件：每批次抽检≥3% + 关注差评关键词变化

**QC与差评联动**：
- 差评中出现「质量差」「做工粗糙」「与描述不符」→ 立即启动全检
- 连续5条以上同类差评 → 暂停该SKU销售排查问题

**关键QC项**（高差评风险）：
| 品类 | 高频差评原因 | QC重点 |
|------|-----------|--------|
| 服装 | 尺码不符/色差/线头 | 尺码表校验/实物拍摄对比/做工 |
| 3C | 功能异常/配件缺失 | 通电测试/配件清单对点 |
| 食品 | 保质期/包装漏气 | 生产日期/密封性 |
| 家居 | 尺寸不符/安装困难 | 安装测试/尺寸测量 |

| BCG象限 | 对应角色 | 策略 |
|---------|---------|------|

## 9. 多轮诊断协议（路径B核心）

**CTR追问链**：渠道(直通车/引力魔方/搜索) → 行业均值(3-5%/2-4%/1-2%) → 词问题还是人群问题 → 归因+变更

| 指标 | 必问第一问 |
|------|----------|
| 转化率低 | 访客质量低还是详情页说服力不足？ |
| ROI低 | 获客成本高还是客单价/复购率低？ |
| DSR下降 | 哪个维度：描述/服务/物流？ |
| 复购率低 | 有没有触达机制？触达了有没有转化？ |

## 10. 复盘标准五问

1. 哪个环节数据最差？曝光/点击/加购/下单/付费？
2. 原因更像流量质量、详情页、还是价格问题？
3. 下个月优先改哪个变量？主图/标题/定价/详情/投放？
4. 本月投放ROI和CAC？有没有超预算渠道？
5. 这次复盘可复用的经验是什么？

## 14. WBS结构

```
电商运营项目
├── 0. 启动（章程+干系人）
├── 1. 规划（市场/竞品/毛利/库存/战略/热词/视觉/定价/漏斗/GMV/搜索/多平台）
├── 0 (前置) 产品开发（可选，仅自有品牌/ODM/OEM）
│   ├── 概念验证（需求/付费意愿/差异化）
│   ├── MVP原型（外观/功能/市场验证）
│   ├── 工艺决策（3D打印vs开模vs钣金vsCNC）
│   └── 供应商寻源与管理（评估+验厂+合同）
├── 0.5 经营架构（五维坐标+NSM+矩阵+BCG）
├── 1. 规划（市场/竞品/毛利/库存/战略/热词/视觉/定价/漏斗/GMV/路线图/搜索/多平台）
├── 2. 执行
│   ├── 2.0 GTM上市SOP（平台选择/定价策略/首波用户/Day1-7时间线）
│   ├── 2.0b QC检查清单（首批全检/持续抽检/差评联动）
│   ├── 2.1 视觉 2.2 上架 2.3 投放 2.4 媒体分发
│   ├── 2.5 私域 2.6 多平台协同 2.7 新品试销
│   ├── 2.8 A/B Test 2.9 信任建设
├── 3. 监控（漏斗/品类+BCG/库存/客户/数据/售后/信任）
└── 4. 收尾（复盘+SOP迭代）
```

## 15. 错误处理

| 场景 | 处理 |
|------|------|
| 数据低于行业均值 | 启动多轮诊断协议 |
| 定价无法覆盖成本 | 重制定价或调产品结构 |
| 投放超预算 | 调渠道/人群/出价 |
| ROI<1持续3天 | 暂停→修转化率→重启 |
| 完整表 → `references/error-handling.md` |
