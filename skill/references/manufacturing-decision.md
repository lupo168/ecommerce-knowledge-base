# 生产工艺决策

> **适用场景**：用户说「开模多少钱」「3D打印还是开模」「怎么做量产」「钣金还是注塑」
> **适用范围**：自有产品进入量产阶段。纯贸易/拿货可跳过。
> **加载条件**：用户已确认产品设计并要求评估量产方案

---

## 一、工艺选择决策树

```
产品外观/结构已确认
│
├─ 数量 < 100件 ────→ 3D打印/CNC/激光切割
│   ├─ 塑料件 → SLA/FDM 3D打印
│   ├─ 金属件 → CNC加工
│   └─ 钣金件 → 激光切割+折弯
│
├─ 数量 100-1000件 ──→ 硅胶复模 / 小批量注塑
│   ├─ 塑料件、外观要求高 → 硅胶复模(¥30-100/件)
│   ├─ 塑料件、结构要求高 → 小批量注塑(模具¥5000-20000)
│   └─ 金属件 → CNC + 简易冲压模
│
└─ 数量 > 1000件 ──→ 开模量产
    ├─ 塑料件 → 注塑模具(¥20000-100000+)
    ├─ 钣金件 → 冲压模具(¥5000-30000)
    ├─ 压铸件 → 压铸模具(¥30000-150000+)
    └─ 硅胶件 → 硫化模具(¥3000-15000)
```

## 二、核心工艺对比

### 3D打印（增材制造）

| 工艺 | 材料 | 精度 | 表面 | 单价(100件) | 适合 |
|------|------|------|------|-----------|------|
| FDM | PLA/ABS/PETG/TPU | ±0.2mm | 粗糙(层纹) | ¥30-150 | 结构件、大件、功能原型 |
| SLA | 光敏树脂 | ±0.1mm | 光滑 | ¥50-200 | 外观件、精细模型 |
| SLS | 尼龙PA12 | ±0.15mm | 磨砂 | ¥80-300 | 功能件、小批量直接使用 |
| MJF | 尼龙PA12 | ±0.1mm | 磨砂 | ¥60-250 | SLS升级版、力学性能更好 |

**3D打印的隐藏成本**：
- 支撑拆除 + 打磨 → 人工成本不可忽略
- 树脂件长期暴露会变黄变脆
- FDM层间结合力弱，受大力会分层断裂

### 开模注塑

| 模具类型 | 价格 | 寿命 | 交期 | 适合 |
|---------|------|------|------|------|
| 简易模(铝模) | ¥5000-20000 | 5000-10000次 | 10-20天 | 试产/小批量 |
| 钢模(标准) | ¥20000-80000 | 10万-30万次 | 25-40天 | 量产 |
| 钢模(高光/镜面) | ¥50000-150000+ | 20万+ | 35-60天 | 外观要求极高 |
| 双色/包胶模 | ¥80000-250000+ | 20万+ | 40-60天 | 双色/软胶包硬胶 |

**开模前的设计检查**（Design for Manufacturing）：
- [ ] 壁厚均匀（薄厚不均→缩水变形）
- [ ] 脱模斜度 ≥ 1°（无斜度→无法脱模或拉伤）
- [ ] 无倒扣（或预留滑块/斜顶位置）
- [ ] 筋位厚度 ≤ 主体壁厚的60%（防止缩水）
- [ ] 浇口位置不破坏外观面
- [ ] 分模线位置可接受

### 钣金加工

| 工艺 | 适用材料 | 厚度范围 | 最小批量 |
|------|---------|---------|---------|
| 激光切割 | 不锈钢/冷轧板/铝板 | 0.5-20mm | 1件 |
| 数控冲床 | 冷轧板/镀锌板 | 0.5-6mm | 50件+ |
| 折弯 | 金属板材 | 0.5-10mm | 1件 |
| 焊接 | 金属件 | - | 1件 |
| 冲压(模具) | 金属板材 | 0.2-6mm | 1000件+ |

**钣金设计注意事项**：
- 折弯半径 ≥ 板厚（更小会开裂）
- 孔距边缘 ≥ 1.5×板厚
- 焊接处预留0.2-1mm间隙
- 表面处理（喷塑/电镀/拉丝）→ 询价时明确

## 三、工艺决策的经济模型

### 盈亏平衡点计算

**3D打印 vs 开模**：
```
设：3D打印单价 = P3d，开模模具费 = M，开模后单价 = Pm
盈亏平衡数量 N = M / (P3d - Pm)

例：SLA打印 ¥80/件，开模 ¥30000 + 注塑 ¥3/件
N = 30000 / (80-3) = 390件
→ 如果总需求 < 390件，3D打印更便宜
→ 如果总需求 > 390件，开模更便宜
```

**CNC vs 冲压**：
```
同理，CNC适合<200件，冲压模具适合>1000件，200-1000之间看交期
```

### 隐藏成本清单

| 成本项 | 典型金额 | 易忽略度 |
|--------|---------|---------|
| 模具试模费 | ¥1000-3000/次 | 高 |
| 模具修改费 | 模具费的10-30% | 高 |
| 注塑调机费 | ¥500-1500/次 | 高 |
| 表面处理（丝印/电镀/喷油） | ¥2-15/件 | 中 |
| 超声波焊接/热熔 | ¥0.5-3/件 | 中 |
| 包装开模（吸塑/纸托） | ¥3000-15000 | 高 |
| 认证费（3C/FCC/CE/ROHS） | ¥5000-50000+ | 高 |
| 模具保养/存储 | ¥500-2000/年 | 中 |

## 四、供应商选择前确认清单

*注：详细供应商管理见 `references/supplier-management.md`*

**询价时必须提供的信息**：
- 3D图纸（STP/IGS格式，不要给STL）
- 2D工程图（含公差标注）
- 材质要求（具体到牌号，如PC+ABS FR3010）
- 表面处理要求（如：咬花MT-11010、喷油Pantone 485C、电镀铬）
- 数量（试产多少+量产预计年用量）
- 交期要求

**比价时的隐藏差异项**：
- A家报价含税含运费但模具费高 → B家报价不含税 → 看似A便宜实际不一定
- 要求供应商报「含税含运费含包装」的总价
- 首批试产价 ≠ 量产后单价，提前锁定量产后阶梯价
