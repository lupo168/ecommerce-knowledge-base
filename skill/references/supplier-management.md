# 供应商寻源与管理

> **适用场景**：用户说「找供应商」「怎么选工厂」「供应商评估」「供应商不靠谱」
> **适用范围**：自有品牌/OEM/ODM。拿货/一件代发可直接用1688筛选。
> **加载条件**：用户需要从零建立供应链或更换供应商

---

## 一、供应商寻源渠道

| 渠道 | 适合 | 优势 | 劣势 |

## 二、供应商评估框架（5维打分）

| 维度 | 权重 | 考察方式 | 评分标准 |
|------|------|---------|---------|
| **质量** | 30% | 打样+验厂+看产线 | 1-10分（10=超出预期） |
| **交期** | 25% | 问已有客户+试单验证 | 准时率>95%=10分 |
| **价格** | 20% | 三家以上比价（含税含运费统一口径） | 中位数以下=8分+ |
| **配合度** | 15% | 沟通响应速度+修改意愿 | 24h内响应=8分+ |
| **资质** | 10% | 营业执照/ISO/品类认证(3C/FCC/CE/ROHS) | 每缺关键资质-2分 |

**一票否决项**：
- 打样三次仍达不到要求
- 无营业执照或经营异常
- 沟通中频繁失信（答应的事不做到）
- 交期承诺离谱（明显低于行业正常交期 → 要么撒谎要么偷工减料）

## 三、供应商分级管理

| 级别 | 定义 | 管理策略 |
|------|------|---------|
| **S级（战略）** | 综合评分≥8.5，核心品类唯一/主供 | 深度绑定：年度框架协议+季度对账+联合开发新品 |
| **A级（主力）** | 综合评分7.0-8.4 | 稳定合作：月度对账+季度复盘 |
| **B级（备用）** | 综合评分5.5-6.9 | 保持联系：半年一次试单，防止供应中断 |
| **C级（淘汰）** | 综合评分<5.5或有否决项 | 清退：清完尾款后终止合作 |

**铁律：任何核心物料/SKU至少有1个A级+1个B级供应商。单一供应商永远脆弱的。**

## 四、供应商合同关键条款

> ⚠️ 以下为框架建议，正式合同请咨询法律专业人士。

| 条款 | 建议内容 |
|------|---------|
| **质量标准** | 明确检验标准（如AQL 2.5）、封样留存、不合格品处理方式 |
| **交期条款** | 交期 ±3天缓冲约定，延迟违约金（每天0.1-0.5%货款）、不可抗力例外 |
| **付款方式** | 建议：30%定金+70%发货前付清。避免100%预付。月结需建立信任后 |
| **知识产权** | 你的图纸/设计的知识产权归你，禁止供应商用于其他客户（OEM/ODM必写） |
| **保密条款** | 约定保密范围（设计图/BOM/成本/客户信息）、保密期限 |
| **模具所有权** | 你付了模具费→模具归你，合同结束可提走或销毁。模具不出售或转用给第三方 |
| **竞业限制** | 供应商不得在合同期内接同类竞争品牌订单（核心供应商） |
| **争议解决** | 约定管辖地（你的所在地），仲裁优先 |

## 五、验厂清单（首次合作必做）

**文件核查（办公室）**：
- [ ] 营业执照、对公账户
- [ ] ISO/3C/FCC/CE/ROHS等证书原件
- [ ] 工人社保/工资发放记录（判断是否正规）
- [ ] 去年产能数据和主要客户名单

**产线核查（车间）**：
- [ ] 产线是否在正常运转（不是空产线做样子）
- [ ] 关键设备的品牌/型号/数量（判断真实产能）
- [ ] 质检流程：有没有质检台、质检标准和记录
- [ ] 物料管理：仓库是否整洁、先进先出是否执行
- [ ] 工人的作业熟练度和精神状态

**打样验证**：
- [ ] 第一次打样：看供应商理解需求的能力
- [ ] 第二次打样：看改进意愿和执行精度
- [ ] 第三次打样：应该已达到量产标准，否则换供应商

## 六、物流发货SOP

*注：Step 14「物流打包发货」属于弱覆盖，此处补充。*

### 物流承运商选择

| 场景 | 推荐 | 价格 | 时效 |
|------|------|------|------|
| 轻小件(电商零售) | 中通/圆通/韵达/极兔 | ¥3-8/件(有量价) | 2-5天 |
| 中大件/贵重 | 顺丰/京东 | ¥12-30+/件 | 1-3天 |
| B2B大宗 | 德邦/安能/跨越 | 按重量体积 | 2-7天 |
| 跨境小包 | 云途/燕文/4PX | ¥15-50/件 | 7-20天 |
| FBA/海外仓 | 以星/美森/铁路 | 按立方米 | 15-45天 |

### 打包SOP
1. 产品外观检查（QC已做，打包前快速扫一眼→明显瑕疵隔离）
2. 配件清单对点（说明书×1/充电线×1/配件包×1）
3. 包装盒/袋封口（易碎贴/防拆贴）
4. 填充物放足（晃动测试：拿起箱子摇3下→内部不应移动）
5. 快递面单粘贴（遮盖地址栏隐私）
6. 按快递公司分堆码放

### 发货时效
- 16:00前订单当日发货（行业标准）
- 预售商品：发货时效标注在SKU标题和详情页
- 异常天气/疫情/春节延迟→客服主动通知，不要等用户来问
