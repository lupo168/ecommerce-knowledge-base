# 视觉分析框架

## 1. 主图分析（5张图分工）

### 1.1 五张图标准分工

| 图片序号 | 类型 | 核心功能 | 设计要点 |
|---------|------|---------|---------|
| **第1张** | 白底图 | 搜索结果页主图 | 正面全貌、清晰展示、品牌logo位置固定 |
| **第2张** | 场景图 | 生活化展示 | 真实使用场景、带入感强 |
| **第3张** | 卖点图 | 差异化展示 | 核心卖点可视化、图文结合 |
| **第4张** | 细节图 | 做工/材质特写 | 局部放大、品质感展示 |
| **第5张** | 营销图 | 活动/促销/背书 | 活动信息、品牌背书、促销力度 |

### 1.2 各平台主图规范

| 平台 | 尺寸 | 格式 | 特殊要求 |
|------|------|------|---------|
| 淘宝 | 800×800px | JPG/PNG | 白底图必须无logo/水印 |
| 天猫 | 800×800px | JPG/PNG | 需品牌授权标识 |
| 京东 | 800×800px | JPG | 第三张图不得含促销信息 |
| 拼多多 | 800×800px | JPG | 可添加场景化背景 |
| 抖音 | 3:4（推荐） | JPG | 可添加字幕/动效 |

## 2. 详情页分析

### 2.1 详情页标准结构

```
┌──────────────────────────────────────────┐
│  ① 痛点引入                                │
│     用户画像 + 使用痛点共鸣                  │
├──────────────────────────────────────────┤
│  ② 解决方案                                │
│     产品是什么、能解决什么问题               │
├──────────────────────────────────────────┤
│  ③ 成分/技术解析                            │
│     核心成分/技术/材质说明                   │
├──────────────────────────────────────────┤
│  ④ 效果对比                                │
│     使用前后对比（真实/合规）                │
├──────────────────────────────────────────┤
│  ⑤ 背书证明                                │
│     资质证书/检测报告/用户好评              │
├──────────────────────────────────────────┤
│  ⑥ 使用方法                                │
│     步骤说明/注意事项                       │
├──────────────────────────────────────────┤
│  ⑦ 规格说明                                │
│     SKU选择指南                            │
├──────────────────────────────────────────┤
│  ⑧ FAQ                                    │
│     常见问题解答                            │
├──────────────────────────────────────────┤
│  ⑨ 售后保障                                │
│     退换货/保修/服务承诺                    │
└──────────────────────────────────────────┘
```

### 2.2 详情页停留时长优化

| 问题 | 优化方向 |
|------|---------|
| 跳出率高 | 优化首屏痛点引入，增加代入感 |
| 停留时间短 | 增加图文结合，减少纯文字段落 |
| 加购率低 | 优化SKU展示，增加紧迫感 |
| 支付率低 | 简化购买路径，增加信任背书 |

## 3. 视觉决策检查清单

### 3.1 主图检查

- [ ] 第1张白底图：正面全貌、清晰、符合平台规范
- [ ] 第2张场景图：真实生活场景、情感共鸣
- [ ] 第3张卖点图：差异化明显、信息传递清晰
- [ ] 第4张细节图：做工/材质/工艺展示
- [ ] 第5张营销图：活动信息准确、品牌背书有力
- [ ] 5张图整体视觉风格统一
- [ ] 没有夸大宣传/违规用词
- [ ] 不同SKU主图有差异化（颜色/规格）

### 3.2 详情页检查

- [ ] 首屏能抓住用户注意力（痛点/利益点）
- [ ] 产品核心卖点突出（1-3个最佳）
- [ ] 有真实买家秀/评价截图
- [ ] 有资质/检测报告背书
- [ ] 使用方法/注意事项清晰
- [ ] SKU选择指南明确
- [ ] 退换货/售后保障信息完整
- [ ] 移动端阅读体验良好（字号/行间距）
- [ ] 加载速度正常（图片压缩）
- [ ] 没有夸大/虚假宣传（合规要求）

## 4. 视觉差异化方向

| 维度 | 差异化方向 |
|------|----------|
| **色调** | 竞品用暖色→我用冷色；竞品用深色→我用浅色 |
| **构图** | 竞品用模特→我用白底；竞品用摆拍→我用场景 |
| **文案** | 竞品写功能参数→我写用户利益 |
| **排版** | 竞品堆字→我用图表；竞品多图→我多文字 |
| **风格** | 竞品走性价比路线→我走品质路线 |
