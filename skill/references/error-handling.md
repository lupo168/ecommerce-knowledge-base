# 错误处理表

| 场景 | 诊断 | 处理 |
|------|------|------|
| 数据低于行业均值 | 漏斗节点异常 | 启动多轮诊断协议 |
| 竞品数据缺失 | 来源受限 | AI推理+标注估算 |
| 定价无法覆盖成本 | 毛利为负 | 重制定价/调产品结构 |
| 投放超预算 | CPI/CAC过高 | 调渠道/人群/出价 |
| 漏斗数据不准 | 数据源问题 | 排除刷单/异常流量 |
| SOP效果不及预期 | 方案环境不匹配 | 复盘五问→调整 |
| 产品资料不全就上架 | 基础未打好 | 强制走难题一SOP |
| ROI<1持续3天 | 产品/定价/页面 | 暂停→修转化率→重启 |
| 人工介入率>50% | 话术不足 | 补充话术+重梳规则 |
| 上架后0曝光 | 收录异常/类目偏差 | 检查类目/属性→重审 |
| 账面盈利现金断流 | 忽略隐性成本 | 三层毛利+库存体检 |
| 多店数据冲突 | 库存/价未同步 | 三同步(价/库存/内容) |
| 私域沉淀<10% | 引导机制缺失 | 查包裹卡+客服话术 |
| 差评突然爆发 | DSR单维下滑 | 差评5步+追责+预警 |
| 分析结论矛盾 | 口径不一致 | 统一口径+核查 |
| 问大家大量负面 | 产品/描述问题 | 修产品→邀满意用户→优详情页 |
| 搜索排名持续降 | 销量/DSR/竞品 | 逐一排查权重因子 |
| 包邮后转化无提升 | 运费非核心因素 | 分析真实决策因素 |
| 裂变参与率低 | 奖励/门槛问题 | 降低门槛+提奖励+优化话术 |
