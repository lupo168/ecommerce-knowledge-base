# 数据分析能力参考

## 数据分析五步法

1. 看趋势(同比/环比): 偶发波动还是持续下滑？
2. 做对比(自身vs行业vs竞品): 在行业什么水平？
3. 拆维度(渠道/人群/时段/地域): 全局还是局部？
4. 找关联(先导→结果): 曝光↓→点击↓→加购↓→成交↓
5. 出方案(定变量+定动作+定预期): 变更请求

## 核心指标速查

| 指标 | 健康 | 预警 | 诊断方向 |
|------|------|------|---------|
| CTR(直通车) | 3-5% | <2% | 词/图/人群必有一错 |
| CTR(引力魔方) | 2-4% | <1% | 人群包不精准 |
| 加购率 | 8-15% | <5% | 详情页说服力不足 |
| 下单转化率 | 3-8% | <2% | 价格/促销感知弱 |
| 退货率 | 5-15% | >20% | 品质/描述不符 |
| ROI(付费) | >=2.5 | <1.5 | CAC过高或客单价太低 |

## 数据仪表盘

**每日**: GMV环比/付费vs预算/主推款转化率/竞品排名
**每周**: 周GMV达成率/各渠道ROI/库存周转/私域新增
**每月**: 月度净利/三层毛利率趋势/复购率/品牌搜索量
