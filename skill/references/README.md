# references/README.md

## 目录说明

本目录包含电商全链路运营工作流的详细方法论文档，按模块划分：

| 文件 | 内容概要 |
|------|---------|
| `pmp-workflow.md` | PMP五过程组与电商运营的完整映射，含WBS/RACI/EVM工具说明 |
| `sales-funnel.md` | 销售漏斗六层详解、EVM绩效指标、监控SOP、异常诊断 |
| `strategy-matrix.md` | 产品/店铺/品牌矩阵推导方法，含四象限模板 |
| `pricing-system.md` | 五价联动定价体系、定价公式、毛利测算方法 |
| `keyword-framework.md` | 热词分类框架、组词公式、全媒体词库布局 |
| `visual-analysis.md` | 主图5张分工设计、详情页结构、视觉决策检查清单 |
| `competitor-analysis.md` | 竞品三维拆解（店铺/单品/SKU）、SWOT分析模板 |
| `media-strategy.md` | 8大平台词库策略、内容种草节奏、达人合作SOP |

使用方式：参考 `assets/` 中的模板文件，结合本目录的方法论，填充具体数据后形成完整运营方案。
