# GMV预估与预算参考

## 推导链路

竞品数据(30天销量)→销量预估(日均x30)→GMV预估(销量x定价)→备货量(日均x补货周期+安全库存)→推广预算(GMVx占比)→盈亏分析

## 销量预估

月销量 = 日均销量 x 30天
日均销量 = 竞品中位数月销量 / 30天

**新店冷启动三阶段**：
1. 达到平均线: 竞品中位数
2. 搜索前48位: 稳定日销
3. 前10位: 头部排名

空白带处理: 相邻低价区间中位数 x 0.3~0.5

## GMV预估

月GMV = 月销量 x 单价

## 备货量

备货量 = 日均销量 x 补货周期 + 安全库存
冷启动安全库存: 0.5~1天销量(补货周期<=7天)

## 推广预算

| 阶段 | 占比 | 说明 |
|------|------|------|
| 冷启动 | GMV x 25% | 以量换排名 |
| 日常 | GMV x 20% | 稳定流量 |

## 盈亏分析

冷启动: 不看盈亏，只看流量+排名
日常运营:
```
月净毛利 = 月GMV x 净毛利率 - 月固定成本
净毛利率 = 1 - 产品成本率 - 平台扣点率 - 推广占比 - 运费包装率
```

## 标准流程

1. 读取竞品数据→月销量
2. 月GMV = Σ(月销量x单价)
3. 备货量+首批成本
4. 推广预算(冷启动25%)
5. 汇总输出(销量/GMV/备货/预算表)
