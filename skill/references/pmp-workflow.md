## 7. 核心工具详解

### 7.1 WBS分解原则

- 每项工作包可分配给单一责任人
- 每个工作包可在80小时内完成
- 遵循 MECE 原则（相互独立，完全穷尽）

### 7.2 RACI矩阵应用

R = 负责（Responsible）执行工作
A = 审批（Accountable）最终责任
C = 咨询（Consulted）需要输入
I = 知会（Informed）需要通知

### 7.3 EVM挣值管理

| 指标 | 公式 | 判断标准 |
|------|------|---------|
| CV（成本偏差） | EV - AC | > 0 省钱，< 0 超支 |
| SV（进度偏差） | EV - PV | > 0 提前，< 0 落后 |
| CPI（成本绩效） | EV / AC | > 1 效率高，< 1 效率低 |
| SPI（进度绩效） | EV / PV | > 1 进度快，< 1 进度慢 |

**电商映射**：
- PV（计划价值）= 计划GMV / 计划流量
- EV（挣值）= 实际GMV / 实际流量
- AC（实际成本）= 实际投放 + 运营成本
