# 定价模型参考

## 三层毛利率计算

```
层次一：账面毛利率 = (售价 - 进货价) / 售价 x 100%
层次二：平台实收毛利率 = (售价 - 进货价 - 平台扣点 - 运费 - 包装) / 售价 x 100%
层次三：真实净利润率 = (售价x(1-退货率) - 进货价 - 平台扣点 - 运费 - 包装 - CAC) / 售价 x 100%
```

## 各平台成本结构参考

| 成本项 | 淘宝/天猫 | 京东 | 拼多多 | 抖音 |
|-------|---------|------|-------|------|
| 平台扣点 | 2-5% | 5-8% | 0.6-2% | 2-5% |
| 运费均摊 | 3-8元/单 | 5-10元/单 | 2-5元/单 | 3-8元/单 |
| 包装成本 | 2-5元/单 | 3-8元/单 | 1-3元/单 | 2-5元/单 |
| 退货率参考 | 5-15% | 3-10% | 10-25% | 15-35% |

## 毛利诊断速查

| 账面毛利率 | 平台实收毛利率 | 判断 | 行动 |
|----------|-------------|------|------|
| >=50% | >=35% | 健康 | 可放量投放 |
| 30-50% | 15-35% | 警戒 | 审视隐性成本 |
| <=30% | <=15% | 危险 | 不整改不能投流 |

> 核心原则：投放前必须先算三层毛利。账面毛利率低于30%的品类，不建议做付费投放。

## 五种定价模型

| 模型 | 计算方式 | 适用场景 | 风险 |
|------|---------|---------|------|
| 成本加成 | 成本 x (1+目标毛利率) | 标品/成本稳定 | 忽视竞争需求 |
| 渗透定价 | 低于市场均价10-20% | 冷启动/抢份额 | 前期亏损/定位低端 |
| 撇脂定价 | 高于均价30-50% | 品牌锚点/技术壁垒 | 需强品牌支撑 |
| 竞品参考 | 取竞品中位数±5% | 常规利润款 | 易陷入价格战 |
| 五价联动 | 标价/划线价/活动价/会员价/渠道价 | 多店铺多平台 | 需协同管理 |

## 库存健康度三色模型

- 绿色(周转15-30天): 正常销售，按周补货
- 黄色(周转30-60天): 启动促销/跨店调拨/达人清仓
- 红色(周转>60天): 立即折价清仓，不计毛利，只求回款

## 现金流三条生命线

| 生命线 | 含义 | 警戒阈值 |
|-------|------|---------|
| 红线(死亡线) | 可用现金=0，货款未回 | 7天内无回款即断流 |
| 黄线(警戒线) | 可用现金<未来30天应付 | 停止所有新采购 |
| 绿线(安全线) | 可用现金>未来60天应付 | 可正常运营 |

## 六项库存指标

| 指标 | 预警阈值 | 触发动作 |
|------|---------|---------|
| 库存周转天数 | >45天 | 促销清仓 |
| 缺货预警 | 连续2天 | 紧急补货 |
| 高库存预警 | >60天TOP5 SKU | 降价/捆绑 |
| 滞销品 | 30天<=3件，占比>20% | 下架/替换 |
| 售罄率 | >90%补货，<30%促销 | 按品类判断 |
| 退货率 | >10% | 排查产品质量 |

## 库存优化决策树

```
库存周转天数 > 45天？
  +- 是 -> 是否缺货？
  |      +- 是 -> 紧急补货(安全库存x2)
  |      +- 否 -> 滞销分析
  |             +- 30天销量<=3件 -> 下架/清仓
  |             +- 30天销量4-10件 -> 促销
  |             +- 30天销量>10件 -> 正常监控
  +- 否 -> 库存健康
```

> 核心原则：宁可断货，不可断流。缺货损失销量，断流损失公司。

## 定价心理学（新增）

### 锚定效应
- 展示「原价¥199，现价¥99」→ 消费者以199为锚，99感知为便宜
- 在详情页前几屏展示高价SKU → 后续中价SKU显便宜
- ⚠️ 划线价必须真实存在过（平台合规要求）

### 价格尾数心理学
| 尾数 | 心理暗示 | 适用场景 |
|------|---------|---------|
| .9 (¥19.9) | 「不到20」便宜感 | 引流款/促销款 |
| .8 (¥198) | 「吉利」中国文化 | 礼品/节日款 |
| .0 (¥200) | 「品质感」整数价 | 品牌锚点款/高端 |
| .5 (¥19.5) | 「合理」「经过计算」 | 利润款 |

### 折中效应
- 提供3个价格梯度时，大多数人选中间
- 策略：设置目标利润款为「中间档」，左右各放一个「更贵/更便宜」做锚
- 例：¥99引流款 / ¥159利润款(目标) / ¥249锚点款

### 损失厌恶
- 「限时折扣倒计时」「库存仅剩X件」唤起「不买就亏了」
- 优惠券设置有效期（领券后24h失效 > 7天失效）
- ⚠️ 平台禁止虚假紧迫感（库存真实，倒计时真实）

### 组合定价
- 买2件8折 vs 第2件半价 → 后者感知「便宜更多」（实际折扣一样）
- 套装定价：A+B组合价 < A+B单独买→引导多件购买提升客单价
