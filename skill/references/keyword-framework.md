# 热词分析框架

## 1. 输入输出

| 要素 | 内容 |
|------|------|
| **输入** | 产品名称 / 产品图 |
| **输出** | Excel词表 + 宝贝名称参考 |

## 2. 词的分类体系

### 2.1 按用途分类

| 词类型 | 定义 | 用途 | 平台 |
|--------|------|------|------|
| **核心词** | 产品本质属性 | 标题主体，必含 | 全平台 |
| **属性词** | 规格/材质/功能/场景 | 精准匹配搜索意图 | 全平台 |
| **营销词** | 促销/品牌/背书 | 吸引点击，提升CTR | 淘宝/天猫 |
| **长尾词** | 核心词+属性词组合 | 精准流量，竞争度低 | 全平台 |
| **竞品词** | 竞品品牌/型号词 | 截流竞品流量（合规） | 全平台 |
| **直通车词** | 付费投放参考词 | 精准出价，控制PPC | 直通车后台 |

### 2.2 按平台分类

| 平台 | 核心词类型 | 特点 |
|------|-----------|------|
| 淘宝/天猫 | 搜索词、标题词根 | 搜索量大，转化率高 |
| 抖音 | 热榜词、挑战赛词 | 热度变化快，内容驱动 |
| 小红书 | 笔记关键词、博主种草词 | 社区氛围，适合种草 |
| 京东 | 搜索词、企业采购词 | B端/C端兼顾 |
| 拼多多 | 活动词、拼单价词 | 价格敏感，活动驱动 |
| 知乎 | 问答词、测评词 | 深度内容，口碑建设 |
| B站 | 测评词、科普词 | 长视频，适合深度测评 |
| 微信 | 搜一搜词、公众号词 | 私域流量 |

## 3. 组词公式

### 3.1 高点击率标题公式

```
标题 = 核心词 + 属性词A + 属性词B + 营销词 + 长尾词A + 长尾词B
```

**示例（净水器）**：

| 词类型 | 词 | 说明 |
|--------|---|------|
| 核心词 | 净水器 | 产品本质 |
| 属性词A | 家用厨下式 | 规格/安装方式 |
| 属性词B | RO反渗透 | 技术类型 |
| 营销词 | 直饮净水机 | 强调使用体验 |
| 长尾词A | 净水器 厨房 RO | 精准长尾 |
| 长尾词B | 家用净水器 直饮 | 场景+功能 |

**完整标题**：
`净水器 家用厨下式 RO反渗透 直饮净水机 净水器 厨房 RO 家用净水器 直饮`

### 3.2 属性词扩展方向

| 属性维度 | 示例词 |
|---------|--------|
| **安装方式** | 台式、厨下式、台上式、壁挂式 |
| **技术类型** | RO反渗透、超滤、活性炭 |
| **容量** | 50G、75G、100G、400G |
| **功能** | 直饮、加热、制冷、矿物质 |
| **使用场景** | 厨房、客厅、办公室、全屋 |
| **材质** | 不锈钢、塑料、玻璃 |
| **品牌背书** | 德国技术、美国滤芯、军工级 |
| **促销** | 厂家直销、限时特价、爆款热卖 |

| 数据来源 | 获取内容 | 适用场景 |

## 5. 词表字段说明

| 字段 | 定义 | 用途 |
|------|------|------|
| **词** | 搜索关键词 | 组词参考 |
| **搜索量（日）** | 平均每天搜索次数 | 流量价值判断 |
| **竞争度** | 在这个词下竞争的商品数 | 红海/蓝海判断 |
| **转化率** | 搜索这个词后实际成交比例 | 高转化词优先 |
| **适合场景** | 标题/直通车/内容 | 用词分类 |
| **相关性** | 与本品的匹配程度 | 过滤不相关词 |
| **出价参考** | 直通车建议出价 | 投放参考 |

---

## 6. 选词标准

| 指标 | 优质词标准 | 淘汰标准 |
|------|----------|---------|
| 搜索量 | 日均 ≥ 500 | < 100 |
| 竞争度 | 竞争度 / 搜索量 < 0.5 | > 1 |
| 转化率 | ≥ 品类均值 | < 品类均值50% |
| 相关性 | 强相关（>80%） | 弱相关或无关 |
| PPC | < 品类平均PPC | 高于平均50% |
