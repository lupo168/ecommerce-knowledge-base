# 资产目录说明

本目录包含电商全链路运营工作流所需的标准化模板和工具：

| 文件 | 类型 | 用途 |
|------|------|------|
| `funnel-report-template.xlsx` | Excel | 销售漏斗数据报表，含公式和条件格式 |
| `wbs-template.md` | Markdown | 电商运营项目 WBS 分解标准模板 |
| `raci-template.md` | Markdown | RACI 矩阵模板（运营/设计/供应链/客服） |
| `keyword-report-template.xlsx` | Excel | 热词分析报表，含搜索量/竞争度/转化率字段 |
| `pricing-calculator.xlsx` | Excel | 五价联动定价计算器，含毛利测算 |
| `ecommerce-sop-template.md` | Markdown | 完整 SOP 文档模板，可直接填充使用 |
| `directory-structure.md` | Markdown | 项目目录结构参考 |
| `launch-checklist.md` | Markdown | 上架前检查清单（启动前必读） |

使用建议：
1. 先用 `wbs-template.md` 规划项目范围
2. 用 `raci-template.md` 明确角色分工
3. 用 `launch-checklist.md` 做上架前检查
4. 运营期用 `funnel-report-template.xlsx` 做数据监控
5. 热词分析后用 `keyword-report-template.xlsx` 整理词表
