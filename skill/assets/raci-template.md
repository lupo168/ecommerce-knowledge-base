# RACI矩阵模板 — 电商运营项目

> R = 负责（Responsible）| A = 审批（Accountable）| C = 咨询（Consulted）| I = 知会（Informed）

---

## 角色说明

| 角色 | 简称 | 说明 |
|------|------|------|
| 运营负责人 | 运营 | 统筹全局，主导决策 |
| 运营专员 | 专员 | 执行落地，日常运营 |
| 设计团队 | 设计 | 视觉制作 |
| 供应链 | 供应 | 库存/定价/物流 |
| 客服团队 | 客服 | 售前咨询/售后处理 |
| 投放专员 | 投放 | 付费广告投放 |
| 平台小二 | 小二 | 平台资源/规则对接 |
| 老板/管理层 | 老板 | 最终审批/战略方向 |

## RACI使用说明

1. **每个任务只能有1个 A**（最终责任人）
2. **每个任务可以有多个 R**（执行责任人）
3. **C 和 I 的数量不宜过多**（保持精简）
4. **无参与用 "-"**（不要留空）
5. **RACI 不是分工表**（是责任矩阵，需要配合组织架构图使用）
