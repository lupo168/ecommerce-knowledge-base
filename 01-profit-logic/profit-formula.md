# 电商利润公式

## 表层公式（常见但不完整）

```
GMV = 流量 × 转化率 × 客单价
```

GMV 只衡量规模，不衡量盈利。做大 GMV 不等于赚钱。

## 完整利润公式

```
利润 = (客单价 × 毛利率 - CAC) × 付费用户数 × 复购次数
```

核心原则：**不是做大分子，而是控制分母。** 流量是买来的，CAC 控得住，模型才成立。

## 公式拆解

| 变量 | 含义 | 决定因素 |
|------|------|---------|
| 客单价 × 毛利率 | 每单毛利贡献 | 定价能力、供应链效率 |
| CAC | 单客获客成本 | 投放效率、自然流量占比 |
| 付费用户数 | 付费用户基数 | 流量 × 整体转化率 |
| 复购次数 | 用户重复购买 | 产品力、私域运营 |

## 关键不等式

定价策略毛利率 ≥ 平台扣点 + 物流 + 退货率

付费投放 CAC < 客单价 × 毛利率

引流款目的：带动利润款，不是利润本身
