# 01 电商赚钱逻辑

理解电商运营的底层盈利模型，知道钱从哪里来、从哪里漏走。

## 核心公式速查

```
GMV = 流量 × 转化率 × 客单价                          （表层）
利润 = (客单价 × 毛利率 - CAC) × 付费用户数 × 复购次数    （完整）
```

## 四杠杆优先级

```
资源有限时：复购 > 转化率 > 客单价 > 流量
```
