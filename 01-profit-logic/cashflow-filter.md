# 现金流过滤原则

**现金流是一切策略的过滤器。** 任何方案必须先过现金流测试。

## 三条铁律

### 1. 定价毛利率底线

```
定价策略毛利率 ≥ 平台扣点 + 物流 + 退货率
```

活动价毛利不能低于运营可变成本。低于这条线，卖得越多亏得越多。

### 2. 付费投放 CAC 上限

```
CAC < 客单价 × 毛利率
```

每获取一个付费客户，投放成本不能超过他贡献的毛利。否则是在补贴平台。

### 3. 引流款定位

引流款目的：**带动利润款，不是利润本身。**

引流款可以零毛利甚至负毛利，但必须有明确的利润款承接路径。如果一个引流款背后没有利润款承接，它只是纯粹的亏损。

## 实操检查表

| 检查项 | 判断标准 |
|--------|---------|
| 日常价毛利率 | ≥ 35%（参考值） |
| 活动价毛利底线 | ≥ 平台扣点 + 物流 + 退货率（通常 ≥ 15%） |
| CAC vs 毛利贡献 | CAC < 客单价 × 毛利率 |
| 引流款→利润款承接 | 有明确的关联推荐、升级路径 |
| 库存周转天数 | < 90 天（防止压货吞噬现金流） |
