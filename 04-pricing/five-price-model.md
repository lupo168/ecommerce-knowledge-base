# 五价联动定价体系

## 1. 五价模型

| 价格类型 | 定义 | 用途 | 定价逻辑 |
|---------|------|------|---------|
| 对手价 | 截流竞品流量 | 卡位参考 | 通常低于竞品5-10% |
| 标价 | 标注划线价 | 锚点感知 | 标价 = 日常价 × 1.3~1.5 |
| 日常价 | 日常销售价 | 稳定转化 | 覆盖运营成本 + 合理利润 |
| 活动价 | 平台大促价 | 冲量保级 | 可降至毛利=平台扣点 |
| 大促价 | 超级大促价 | 拉新/清仓 | 低于日常价10-20% |
| 批发价 | 批量采购价 | 分销/达人 | 日常价 × 0.6~0.8 |

## 2. 定价公式

### 2.1 基础毛利率

```
毛利率 = (日常价 - 产品成本) / 日常价 × 100%
```

### 2.2 活动价最低毛利底线

```
活动价毛利 ≥ 平台扣点(5-8%) + 物流(3-5%) + 退货率(5-15%)
```

示例：平台扣点 6%，物流 4%，退货率 8%
- 活动价毛利底线 = 6% + 4% + 8% = 18%
- 若产品成本 80元，活动价最低 = 80 / (1 - 18%) = 97.56元

### 2.3 CAC 上限

```
CAC = 总投放成本 / 付费人数
健康标准：CAC < 客单价 × 毛利率
```

### 2.4 衍生价格公式

```
标价 = 日常价 × (1.3 ~ 1.5)
大促价 = 日常价 × (0.8 ~ 0.9)
批发价 = 日常价 × (0.6 ~ 0.8)
```

## 3. 实操示例（净水器品类）

| 价格类型 | 金额 | 说明 |
|---------|------|------|
| 产品成本 | ¥1,200 | 采购/生产成本 |
| 日常价（毛利40%） | ¥2,000 | 目标毛利 |
| 标价（划线价） | ¥2,999 | 日常价 × 1.5，制造锚点 |
| 活动价 | ¥1,699 | 大促/平台活动，毛利约18% |
| 大促价 | ¥1,499 | 超级大促极限价，毛利约10% |
| 对手价 | ¥1,799 | 截流竞品，略低于竞品 |
| 批发价 | ¥1,300 | 达人/分销批量价，毛利约8% |

## 4. 定价策略检查清单

| 检查项 | 要求 | 判断标准 |
|-------|------|---------|
| 日常价毛利 | ≥ 运营成本 + 合理利润 | 毛利率 ≥ 35% |
| 活动价毛利 | ≥ 平台扣点 + 物流 + 退货 | 毛利 ≥ 15% |
| 对手价卡位 | 低于核心竞品5-10% | 竞品价格可查 |
| 标价锚点 | 标价 ≥ 日常价 × 1.3 | 有划线价展示 |
| CAC上限 | CAC < 客单价 × 毛利率 | 可持续投放 |
| 批发价保护 | 批发价 > 产品成本 | 不得亏本出货 |
