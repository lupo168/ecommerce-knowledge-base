# Phase 1：规划（Planning）

规划阶段是运营成败的关键——分析先行，战略从分析中生长。

## 1.1 市场分析（收集需求）

| 维度 | 分析指标 | 数据来源 |
|------|---------|---------|
| 大盘规模 | GMV/销售额/增长率 | 生意参谋/情报通 |
| 增长趋势 | 近12个月趋势 | 生意参谋 |
| 季节性 | 淡旺季分布 | 历史数据 |
| 消费者画像 | 年龄/性别/地域/消费力 | 生意参谋/用户调研 |
| 政策/规则 | 平台近期变化 | 官方公告 |

## 1.7 风险登记册

| 风险类别 | TOP3风险 | 应对策略 |
|---------|---------|---------|
| 竞争风险 | 竞品大幅降价 | 提前制定反击预案 |
| 平台风险 | 规则突然收紧 | 合规运营，留有缓冲空间 |
| 供应链风险 | 断货/爆品 | 多供应商备份，安全库存 |
| 流量风险 | 付费ROI持续下滑 | 拓宽流量渠道，降低依赖 |
| 资金风险 | 账期拉长/压货 | 控制库存周转，控制账期 |
