# Phase 3：监控（Monitoring & Controlling）

## 核心监控体系

### 漏斗日常监控

```
每日 09:00
├── 查看昨日漏斗数据（曝光/点击/加购/成交）
├── 识别最大流失环节（红标预警）
├── 对比昨日/上周同期
└── 判断：正常波动 / 需干预
```

详见 [销售漏斗分析](../03-analysis-methods/sales-funnel.md)

### EVM 挣值管理（电商映射）

| 指标 | 公式 | 判断标准 |
|------|------|---------|
| CV（成本偏差） | EV - AC | > 0 省钱，< 0 超支 |
| SV（进度偏差） | EV - PV | > 0 提前，< 0 落后 |
| CPI（成本绩效） | EV / AC | > 1 效率高，< 1 效率低 |
| SPI（进度绩效） | EV / PV | > 1 进度快，< 1 进度慢 |

**电商映射**：
- PV（计划价值）= 计划GMV / 计划流量
- EV（挣值）= 实际GMV / 实际流量
- AC（实际成本）= 实际投放 + 运营成本

## 预警机制

| 预警级别 | 触发条件 | 响应时限 | 负责人 |
|---------|---------|---------|-------|
| 🔴 红（严重） | 某层转化率跌破预警阈值 | 2小时内响应 | 运营负责人 |
| 🟡 黄（注意） | 某层转化率连续3天下滑>10% | 24小时内响应 | 运营专员 |
| 🟢 绿（正常） | 各层转化率在健康区间 | 例行监控 | 数据分析 |

## 报告节奏

| 报告 | 频率 | 内容 |
|------|------|------|
| 日报 | 每日 | 销量/流量/转化/付费消耗 |
| 周报 | 每周 | 周环比/竞品动态/下周计划 |
| 月报 | 每月 | 月度复盘/策略调整/SOP更新 |
