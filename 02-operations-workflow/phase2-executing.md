# Phase 2：执行（Executing）

| 电商动作 | PMP过程 | 产出物 |
|---------|---------|-------|
| 市场/竞品调研 | 指导与管理项目工作 | 调研报告（数据/截图/表格） |
| 视觉制作 | 管理质量 | 主图5张 + 详情页 + 视频 |
| 上架运营 | 获取资源 / 建设团队 | 商品上架、任务分工 |
| 付费投放 | 管理沟通 | 投放计划、日/周报 |
| 达人合作 | 实施采购 | 合同签订、档期锁定 |
| 客服运营 | 相关方管理 | FAQ、客诉处理SOP |

## 上架前检查

上架前必须逐项确认十项检查内容。详见 [上架检查清单](../06-templates/launch-checklist.md)

## 执行原则

- **先规划后执行**：不要在缺乏分析支撑的情况下执行
- **最小可行上架**：先上核心 SKU 跑数据，再扩展变体
- **执行中监控**：执行不是闭眼做，同步跑漏斗监控
