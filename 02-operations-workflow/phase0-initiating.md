# Phase 0：启动（Initiating）

## 0.1 立项章程

| 要素 | 内容 | 示例 |
|------|------|------|
| 项目名称 | XX品类电商运营项目 | 「家居品类电商运营」 |
| 运营目标 | GMV目标 / 流量目标 / 转化目标 | 「季度GMV 50万」 |
| 时间周期 | 季度/月度运营周期 | 「Q2（4-6月）」 |
| 资源配置 | 人员分工 / 预算分配 | 「运营1人+设计1人+投放2万/月」 |
| 成功标准 | ROI ≥ X / 转化率 ≥ X% / 客单价 ≥ X元 | 「ROI ≥ 2.5」 |

## 0.2 干系人识别

| 角色 | 职责 | 管理策略 |
|------|------|---------|
| 项目发起人 | 审批预算、确定战略方向 | 定期汇报、数据透明 |
| 运营负责人 | 统筹全局、执行落地 | 赋权执行、快速决策 |
| 设计团队 | 视觉制作（主图/详情/视频） | 提前沟通、避免反复 |
| 供应链 | 库存/定价/物流支撑 | 信息同步、预测前置 |
| 平台小二 | 活动资源、规则解读 | 维护关系、争取资源 |
| 客服团队 | 售前咨询、售后处理 | 培训产品知识、建立FAQ |
