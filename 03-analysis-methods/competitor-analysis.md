# 竞品三维拆解框架

## 1. 三维分析模型

```
竞品分析
├── 1. 店铺层分析（全局视角）
├── 2. 单品层分析（产品视角）
└── 3. SKU层分析（变体视角）
```

## 2. 店铺层分析

### 2.1 店铺基础信息

| 分析维度 | 数据来源 | 说明 |
|---------|---------|------|
| 店铺名称 | 搜索结果 | 品牌露出 |
| 店铺类型 | 店铺首页 | 天猫店/淘宝店/专卖店 |
| DSR评分 | 店铺首页 | 描述/服务/物流 |
| 开店时长 | 店铺首页 | 老店 vs 新店 |
| 粉丝数 | 店铺首页 | 品牌积累 |
| 主营类目 | 店铺首页 | 是否专注 |
| 价格带 | 全店商品 | 高中低分布 |

### 2.2 店铺运营能力

| 分析维度 | 指标 |
|---------|------|
| 动销率 | 在售商品数 / 总商品数 |
| 上新频率 | 月上新商品数 |
| 爆款数量 | 销量>10000的商品数 |
| 活动参与 | 参与平台活动频率 |
| 付费投放 | 直通车/引力魔方投入 |

## 3. 单品层分析

### 3.1 爆款分析表

| 分析项 | 数据获取方式 |
|-------|------------|
| 销量 | 生意参谋/搜索结果 |
| 销售额估算 | 销量 × 单价 |
| 收藏量 | 商品详情页 |
| 评价数 | 商品详情页 |
| 买家秀数量 | 评价详情 |
| 问大家问题数 | 问大家板块 |
| 上新时间 | 历史记录 |

### 3.2 销量来源拆解

| 流量来源 | 判断方法 | 占比参考 |
|---------|---------|---------|
| 搜索流量 | 自然排名位置 | 头部商品 40-60% |
| 推荐流量 | 手淘首页曝光 | 爆款 20-40% |
| 付费流量 | 直通车消耗/销量 | 通常 20-30% |
| 直播流量 | 是否有直播 | 视情况 |
| 私域流量 | 粉丝/老客复购 | 视情况 |

## 4. SKU 层分析

### 4.1 SKU 变体设计分析

| 分析维度 | 内容 |
|---------|------|
| 规格类型 | 颜色/容量/套装/单买/多买 |
| SKU数量 | 总变体数量 |
| 价格梯度 | 各SKU之间价格差 |
| 热销SKU | 销量最高的SKU |
| SKU设计策略 | 单链接 vs 多链接 |

### 4.2 SKU 策略判断

| 策略类型 | 特征 | 适用场景 |
|---------|------|---------|
| 单链接多SKU | 一个链接含多个规格 | 减少流量分散，方便管理 |
| 多链接单SKU | 每个规格单独链接 | 精细化运营，流量精准 |
| 混合策略 | 主SKU单链接，辅规格多链接 | 兼顾精准与效率 |

## 5. SWOT 分析模板

| 维度 | 内容 |
|------|------|
| S（优势） | 自身优势（品牌/供应链/团队/资金） |
| W（劣势） | 自身劣势（知名度/运营经验/产品力） |
| O（机会） | 市场机会（空白价格带/新兴渠道/政策红利） |
| T（威胁） | 外部威胁（竞品强势/平台规则变化/资金链风险） |

**竞品对比 SWOT 示例**：

| 维度 | 自身 | 竞品A | 竞品B |
|------|------|-------|-------|
| S | 供应链成本低 | 品牌知名度高 | 运营团队强 |
| W | 品牌力弱 | 价格偏高 | SKU过多 |
| O | 性价比市场空白 | 消费升级趋势 | 下沉市场需求大 |
| T | 竞品价格战 | 新进入者 | 平台规则收紧 |

## 6. 竞品监控表

| 监控项 | 频率 | 关注指标 |
|-------|------|---------|
| 价格监控 | 每日 | 日常价/活动价变化 |
| 销量监控 | 每日 | 销量排名/销售额排名 |
| 评价监控 | 每日 | 新增差评/问大家负面 |
| 上新监控 | 每周 | 新品上架/爆款更新 |
| 活动监控 | 实时 | 大促/秒杀/直播 |
| 投放监控 | 每周 | 直通车词/投放力度变化 |
