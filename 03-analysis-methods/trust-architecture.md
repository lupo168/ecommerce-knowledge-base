# 信任体系建设

## 信任信号金字塔

```
               ┌──────────────┐
               │  平台担保层    │ 消费者保障/正品险/假一赔十
              ┌┴──────────────┴┐
              │  社交验证层     │ 评价/买家秀/问大家/KOL
             ┌┴────────────────┴┐
             │   品牌权威层      │ 资质认证/检测报告/品牌故事
            ┌┴──────────────────┴┐
            │   交易安全层        │ 运费险/7天无理由/保价
           ┌┴────────────────────┴┐
           │    基础信息层         │ 店铺类型/DSR/销量/发货地
           └──────────────────────┘
```

信任越往下越基础，越往上越强。每一层的缺失都会导致下一层的信任信号被质疑。

## 问大家运营

### 三步 SOP

1. **问题预埋**（上架前）：预设 5-10 个高频问题，邀请首批用户回答
2. **日常监控**（每周）：检查新问题（> 24h 未回复影响转化），发现负面 → 联系用户解决
3. **内容反哺**（每月）：提取高频关键词优化详情页

### 诊断速查

| 症状 | 行动 |
|------|------|
| 问题数 = 0 | 预埋 5-10 个种子问题 |
| 问题多但无回答 | 联系已购用户邀请回答 |
| 负面占主导 | 先解决产品，再邀请满意用户 |
| 大量重复 | 写入详情页 |

## 资质展示策略

| 资质类型 | 展示位置 | 设计要点 |
|---------|---------|---------|
| 检测报告 | 详情首屏或信任区 | 标注关键指标 |
| 获奖证书 | 主图第 5 张或详情 | 标注年份 |
| 专利证书 | 技术说明区 | 标注专利号和创新点 |
| 品牌授权 | 主图或详情首页 | 注意授权期限 |
| 工厂实拍 | 详情或视频 | 真实场景优于精修 |

## DSR 三维管理

| 维度 | 目标 | 核心动作 |
|------|------|---------|
| 描述相符 | ≥ 4.8 | 实物与描述一致性 |
| 服务态度 | ≥ 4.8 | 响应速度 + 专业度 |
| 物流速度 | ≥ 4.7 | 发货时效 + 异常处理 |

## 售后分级响应

| 级别 | 占比 | 类型 | 响应标准 |
|------|------|------|---------|
| L1 | 80% | 退换货/物流查询 | 自动客服 15 秒 → 转人工 → 2h 解决 |
| L2 | 15% | 品质投诉/错发 | 人工 5 分钟 → 补偿 → 24h 闭环 |
| L3 | 5% | 安全/批量 | 主管直管 → 2h 出方案 → 48h 闭环 |

## 差评处理 5 步

1. 发现 → 5 分钟电话联系
2. 倾听道歉补偿
3. 内部追责（产品/仓储/物流/描述）
4. 整改
5. 3 天回访 → 归档

## 好评四步法

超预期交付 → 签收 3 天关怀 → 满意引导好评（提供模板）→ 口碑放大（买家秀素材）

## 价格保价策略

- 7 天/15 天保价：消除等促销心理
- 大促保价：预付定金无需担心降价
- 买贵退差：直接消除比价顾虑
- 不保价（清仓）：制造稀缺感
