# GMV 预估与预算推导

## 推导链路

```
竞品数据(30天销量) → 销量预估(日均×30) → GMV预估(销量×定价) → 备货量(日均×补货周期+安全库存) → 推广预算(GMV×占比) → 盈亏分析
```

## 销量预估

```
月销量 = 日均销量 × 30天
日均销量 = 竞品中位数月销量 / 30天
```

**新店冷启动三阶段**：

| 阶段 | 目标 | 说明 |
|------|------|------|
| 1 | 达到平均线 | 竞品中位数 |
| 2 | 搜索前 48 位 | 稳定日销 |
| 3 | 前 10 位 | 头部排名 |

空白价格带处理：相邻低价区间中位数 × 0.3 ~ 0.5

## GMV 预估

```
月 GMV = 月销量 × 单价
```

## 备货量

```
备货量 = 日均销量 × 补货周期 + 安全库存
冷启动安全库存: 0.5 ~ 1 天销量（补货周期 ≤ 7天）
```

## 推广预算

| 阶段 | 预算占比 | 说明 |
|------|---------|------|
| 冷启动 | GMV × 25% | 以量换排名 |
| 日常 | GMV × 20% | 稳定流量 |

## 盈亏分析

**冷启动**：不看盈亏，只看流量 + 排名

**日常运营**：
```
月净毛利 = 月GMV × 净毛利率 - 月固定成本
净毛利率 = 1 - 产品成本率 - 平台扣点率 - 推广占比 - 运费包装率
```

## 标准流程

1. 读取竞品数据 → 月销量
2. 月 GMV = Σ (月销量 × 单价)
3. 备货量 + 首批成本
4. 推广预算（冷启动 25%）
5. 汇总输出（销量 / GMV / 备货 / 预算表）
