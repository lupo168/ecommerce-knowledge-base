# A/B 测试方法

## 可测维度

| 决策层 | 变量 | 主指标 | 次指标 |
|--------|------|--------|--------|
| 主图 | 白底vs场景/卖点排布 | CTR | 加购率 |
| 标题 | 词顺序/属性组合 | CTR | 搜索排名 |
| 定价 | 价格AvsB/折扣vs券 | 转化率 | 客单价 |
| 详情首屏 | 痛点vs卖点切入 | 加购率 | 停留时长 |
| 促销 | 直降vs满减vs赠品 | 下单率 | 毛利率 |

## 五步流程

1. **假设** — 可证伪，只改一个变量
2. **指标+样本** — 每组 ≥ 500 曝光
3. **执行** — 同期同人群，其他变量冻结
4. **决策** — 差异 < 5% 选低成本 / 5-15% 倾向胜方 / > 15% 全量切换
5. **归档** — 写入经验教训登记册

## 各平台工具

| 平台 | 工具 |
|------|------|
| 淘宝/天猫 | 万相台 AB Test / 直通车创意测试 |
| 抖音 | 千川创意测试 |
| 无工具 | 双链接法 / 优惠券法 / 分时段 |

## 测/不测决策

**测**：两方案各有道理 / 重大资源投入 / 方向模糊

**不测**：样本不足（< 500）/ 大促前后 / 已有历史数据
