# 搜索曝光优化

## 搜索排序算法权重拆解

| 权重因子 | 影响 | 优化方向 |
|---------|------|---------|
| 关键词相关性 (30-40%) | 标题与搜索词匹配度 | 标题词根覆盖 + 属性词精准 |
| 销量权重 (20-30%) | 近 30 天确认收货 | 冷启动促销冲量破零 |
| 转化率 (15-20%) | 曝光 → 点击 → 成交链 | 主图 CTR + 详情说服力 |
| DSR 评分 (10-15%) | 三维评分 | 维持 ≥ 4.8 |
| 店铺权重 (5-10%) | 层级/活跃度/违规 | 持续上新 + 避免扣分 |

## 长尾词挖掘策略

| 词类 | 来源 | 策略 |
|------|------|------|
| 搜索联想词 | 搜索框下拉 | 精准匹配 + 高出价 |
| 长尾属性词 | 竞品标题反查 | 广泛匹配 + 低出价覆盖 |
| 场景词 | 问大家/评价 | 融入标题和详情 |
| 竞品品牌词 | 竞品品牌 + 品类 | 合规截流 |

## 筛选条件优化

| 维度 | 用户常用筛选 | 优化目标 |
|------|------------|---------|
| 价格区间 | 按预算 | 落在主流区间 |
| 发货地 | 按距离 | 多仓覆盖核心区 |
| 品牌 | 按品牌 | 品牌种草 + 词投放 |
| 好评率 | ≥ 4.8 | 维持高 DSR |

## 运费策略

| 策略 | 做法 | 适用 |
|------|------|------|
| 全店包邮 | 免运费 | 高毛利/品牌锚点款 |
| 门槛包邮 | 满 X 元免（门槛 = 客单价 × 1.2） | 低客单/引流款 |
| 单品包邮 | 爆款免 / 长尾不免 | 利润 + 引流组合 |
| 区域包邮 | 仅核心区 | 仓储布局有限 |

## 四阶段破零策略

| 阶段 | 策略 | 节奏 |
|------|------|------|
| Day 1-3 | 促销价 + 首单优惠 + 私域内购 | 快速 10-20 单 |
| Day 4-7 | 加大投放 + 关联推荐 + 直播引流 | 日均稳定出单 |
| Day 8-14 | 评价引导 + 买家秀 | 积累信任信号 |
| Day 15+ | 调定价/主图/投放预算 | 进入日常运营 |

## 视觉竞争策略

| 策略 | 做法 | 场景 |
|------|------|------|
| 色彩跳脱 | 竞品浅色 → 你用深色 | 同质化标品 |
| 构图反常规 | 竞品正面 → 你 45 度或场景 | 视觉区分度低 |
| 文字信息差 | 竞品标价 → 你标卖点 | 功能型产品 |
| 场景差异化 | 竞品白底 → 你用实景 | 需代入感 |
| 视频壁垒 | 竞品无 → 你抢先 | 任何品类 |

## 竞品对比图 SOP

1. 选 3 个最有力差异（功能/效果/成本）
2. 左旧右新/左竞右我 + 红框标注 + 一句总结
3. 不可贬低竞品 + 数据真实 + 放详情中后段
