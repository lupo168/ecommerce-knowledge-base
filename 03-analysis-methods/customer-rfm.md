# RFM 客户分层模型

## RFM 三维模型

| 维度 | 含义 | 电商场景 |
|------|------|---------|
| R (Recency) | 最近一次购买时间 | 判断活跃度 |
| F (Frequency) | 购买次数 | 判断忠诚度 |
| M (Monetary) | 累计消费金额 | 判断价值 |

## 客户价值分层矩阵

| RFM 组合 | 客户类型 | 运营策略 |
|---------|---------|---------|
| 高R 高F 高M | **重要价值客户** | VIP 维护/专属优惠/优先发货 |
| 高R 低F 高M | **重要发展客户** | 提升频次/会员引导 |
| 低R 高F 高M | **重要保持客户** | 召回激活/专属优惠 |
| 低R 低F 高M | **重要挽留客户** | 深度召回/问题调研 |
| 高R 高F 低M | **潜力客户** | 升级引导/高毛利推荐 |
| 高R 低F 低M | **新客户** | 培育激活/复购引导 |
| 低R 高F 低M | **流失风险客户** | 召回优惠/预警 |
| 低R 低F 低M | **一般客户** | 低成本触达/促销推送 |

## 客户生命周期管理

| 阶段 | 核心动作 | 目标指标 |
|------|---------|---------|
| 获取期 | 新客首单优惠/引流款成交 | 新客转化率 |
| 培育期 | 使用引导/复购激励 | 30天复购率 |
| 成熟期 | VIP 权益/专属推荐 | LTV/复购频次 |
| 衰退期 | 流失预警/召回激励 | 流失率/召回率 |
| 挽回期 | 问题调研/专属补偿 | 挽回率 |

## 客户指标监控

| 指标 | 健康阈值 |
|------|---------|
| 新客户数量 | 持续增长 |
| 新客获客成本 | < 新客首单毛利额 |
| 老客复购率 | > 15% |
| 老客客单价 | > 新客客单价 × 1.5 |
