## 3. 达人合作 SOP

### 3.1 达人筛选标准

| 维度 | 标准 | 权重 |
|------|------|------|
| 粉丝量 | 1万+（素人）/ 10万+（腰部）/ 100万+（头部） | 参考 |
| 内容质量 | 点赞/收藏/评论比 | 核心 |
| 受众匹配 | 粉丝画像与目标用户重合度 | 核心 |
| 带货历史 | 过往带货转化数据 | 核心 |
| 配合度 | 回复速度/修改配合度 | 参考 |

### 3.2 合作流程

```
筛选达人 → 建联沟通 → 确定合作形式 → 签订合同
    ↓
寄样/资料 → 内容审核 → 发布执行 → 数据复盘
```

| 步骤 | 注意事项 |
|------|---------|
| 筛选 | 匹配度 > 粉丝量，优先选精准粉 |
| 建联 | 私信/邮件/MCN，说明合作形式和预算 |
| 合同 | 明确坑位费/佣金/保底，约定数据要求 |
| 寄样 | 产品 + 品牌资料包，附上FAQ |
| 审核 | 内容方向/文案/产品露出，提前沟通脚本 |
| 发布 | 约定发布时间，蹲守评论区 |
| 复盘 | 阅读/点赞/评论/带货数据，留存归档 |

## 4. 内容种草节奏规划

| 时间节点 | 内容动作 | 目的 |
|---------|---------|------|
| 上架前4周 | 知乎/小红书种草铺垫 | 建立口碑基础 |
| 上架前1周 | 预热视频/笔记发布 | 预热造势 |
| 上架当天 | 直播/短视频集中爆发 | 快速起量 |
| 上架后1个月 | 持续种草 + 数据监控 | 维护排名 |
| 日常 | 定期更新 + 竞品词截流 | 持续曝光 |
