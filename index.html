快速开始

从0到1搭建电商运营体系：

  1. 打开 上架检查清单，建立全局视图
  2. 按 Phase 0 完成立项
  3. 使用 竞品分析框架 摸清市场
  4. 制定 产品矩阵 和 定价方案
  5. 搭建 销售漏斗监控

排查具体问题

  • CTR 低 → 销售漏斗诊断追问链
  • 转化率低 → 四大杠杆 — 转化率
  • 定价没把握 → 五价联动模型
  • 竞品打不过 → 竞品三维拆解

核心原则

分析先行，战略从分析中生长 — 严禁拍脑袋定战略再找数据凑理由。
现金流是一切策略的过滤器 — 任何方案必须先过现金流测试。
