# 电商上架前检查清单

> 在商品上架之前，必须逐项确认以下内容。

---

## 一、商业分析完成确认

- [ ] 市场大盘分析已完成（规模/趋势/季节性）
- [ ] 核心竞品已锁定（3-5家）
- [ ] 竞品分析已完成（店铺/单品/SKU三维）
- [ ] SWOT分析已完成
- [ ] 市场机会点已识别

---

## 二、战略矩阵完成确认

- [ ] 产品矩阵已制定（引流/利润/形象/防御）
- [ ] 店铺矩阵已规划
- [ ] 品牌定位已明确（高端/大众/性价比/白牌）
- [ ] 战略有分析数据支撑（不是拍脑袋）

---

## 三、热词分析完成确认

- [ ] 核心词已确定（1-2个）
- [ ] 属性词已扩展（规格/功能/场景/背书）
- [ ] 长尾词已整理（3-5词组合）
- [ ] 高点击率标题已生成（至少3个方案）
- [ ] 直通车投放词已准备
- [ ] 全媒体词库已覆盖8大平台

---

## 四、视觉设计完成确认

### 主图5张
- [ ] 第1张白底图已完成，符合平台规范
- [ ] 第2张场景图已完成，生活化展示
- [ ] 第3张卖点图已完成，差异化明显
- [ ] 第4张细节图已完成
- [ ] 第5张营销图已完成，活动信息准确
- [ ] 5张图整体风格统一
- [ ] 没有违规/夸大宣传内容

### 详情页
- [ ] 详情页结构完整（9大模块）
- [ ] 首屏能抓住用户注意力
- [ ] 产品核心卖点突出（1-3个）
- [ ] 有买家秀/评价截图
- [ ] 有资质/检测报告背书
- [ ] SKU选择指南明确
- [ ] 退换货/售后保障信息完整
- [ ] 移动端阅读体验良好
- [ ] 没有违规宣传内容

---

## 五、定价策略完成确认

- [ ] 五价体系已制定（标价/日常价/对手价/活动价/大促价/批发价）
- [ ] 日常价毛利 ≥ 35%（参考值）
- [ ] 活动价毛利底线验证通过（≥平台扣点+物流+退货）
- [ ] 对手价已低于核心竞品5-10%
- [ ] 标价锚点合理（≥日常价×1.3）
- [ ] 竞品价格对比已完成

---

## 六、商品上架信息确认

### 基础信息
- [ ] 商品标题已按热词分析结果组词
- [ ] 商品属性完整填写
- [ ] SKU规格设置正确（颜色/容量/套装）
- [ ] 价格设置正确（五价联动）
- [ ] 库存数量已确认

### 平台规则
- [ ] 商品类目选择正确
- [ ] 没有违禁词/违禁内容
- [ ] 品牌授权（如需）已准备
- [ ] 产品资质证书已上传（如需）

### 物流/服务
- [ ] 运费模板设置正确
- [ ] 发货时间设置正确
- [ ] 售后服务条款设置正确
- [ ] 退换货政策设置正确

---

## 七、销售漏斗规划确认

- [ ] 各层转化率阈值已定义
- [ ] 漏斗监控报表模板已准备好
- [ ] 预警机制已建立
- [ ] EVM指标（ROI/CAC/LTV）计算公式已确认

---

## 八、投放规划确认

- [ ] 直通车计划已制定
- [ ] 关键词出价策略已确定
- [ ] 人群定向策略已确定
- [ ] 每日预算已分配
- [ ] 投放目标（曝光/点击/ROI）已明确

---

## 九、团队协作确认

- [ ] 客服团队已培训（产品知识/话术/FAQ）
- [ ] 仓库已确认库存充足
- [ ] 物流已确认能按时发货
- [ ] 突发情况应急预案已准备

---

## 十、最后检查

- [ ] 商品链接在PC端预览正常
- [ ] 商品链接在手机端预览正常
- [ ] 价格/库存/SKU信息再次核对
- [ ] 主图/详情页再次检查无误
- [ ] 上下架时间已设置（如需要）

---
